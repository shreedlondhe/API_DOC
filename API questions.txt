API interview questions

Basic JSON Questions
1. What is JSON?

JSON is a lightweight data format used for storing and exchanging data between applications, especially in APIs.

2. What does JSON stand for?

JSON stands for JavaScript Object Notation.

3. What are the supported data types in JSON?

JSON supports:

String
Number
Boolean
Object
Array
Null
4. What is the difference between JSON and XML?
JSON	XML
Lightweight	Heavy
Easy to read	More complex
Faster parsing	Slower
Uses key-value pairs	Uses tags
5. What is the difference between JSON and JavaScript object?

JSON is a text format used for data exchange, while a JavaScript object is an in-memory object used in JavaScript programs.

6. Is JSON case-sensitive?

Yes, JSON keys and values are case-sensitive.

7. Can JSON support comments?

No, JSON does not support comments.

8. What is the difference between null and empty string in JSON?
null means no value
"" means value exists but empty
9. What are key-value pairs in JSON?

JSON stores data as key-value pairs.

Example:

{
  "name": "Shrinivas"
}
10. Can a JSON file have duplicate keys?

Technically possible, but not recommended. Most parsers take the last value.

Structure-Based Questions
11. What is a nested JSON object?

A JSON object inside another JSON object is called nested JSON.

12. What is a JSON array?

A collection of values stored inside square brackets [].

13. Difference between JSON object and JSON array?
Object → uses {}
Array → uses []
14. How do you access nested JSON values?

Using keys.

Example:

{
  "employee": {
    "name": "Shrinivas"
  }
}

Access:

employee.name
15. How do you handle dynamic values in JSON response?

Store values in variables and validate dynamically using JSONPath or assertions.

16. How to parse JSON in Java?

Using libraries like:

Jackson
Gson
17. How to convert JSON string to object?

Using deserialization.

Example:

ObjectMapper.readValue()
18. How to convert object to JSON string?

Using serialization.

Example:

ObjectMapper.writeValueAsString()
19. What is serialization and deserialization in JSON?
Serialization → Object to JSON
Deserialization → JSON to Object
20. What is pretty print JSON?

Formatted JSON with indentation for readability.
